###########################
Gestion des serveurs locaux
###########################

Comme les serveurs locaux de la gateway font partie intégrante de celle-ci,
ajouter/modifier les serveurs sert en fait à gérer la configuration de la gateway.
Une fois démarré, chaque serveur sera considéré comme un service de la gateway.
