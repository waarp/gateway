=========================
Afficher un moniteur SNMP
=========================

.. program:: waarp-gateway snmp monitor get

Affiche les informations du moniteur SNMP donné en paramètre.

**Commande**

.. code-block:: shell

   waarp-gateway snmp monitor get "<NAME>"


**Exemple**

.. code-block:: shell

   waarp-gateway snmp monitor get "nagios"
