#######################
Gestion des traitements
#######################

Le point d'accès pour gérer les traitements d'une règle de transfert est
``/api/rules/<rule_id>/tasks``.

Pour supprimer tous les traitements d'une règle, il suffit de remplacer les 3
chaînes par des listes vides.
