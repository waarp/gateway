Gestion des comptes locaux
##########################

Le point d'accès pour gérer les comptes locaux est ``/api/servers/{nom_serveur}/accounts``.
Un compte local est toujours rattaché à un serveur, et sert donc à s'authentifier
sur ce dernier.
