Gestion des autorités d'authentification
########################################

Le point d'accès pour gérer les autorités d'authentification de la gateway est
```/api/authorities``. Une autorité d'authentification est un tier auquel la
*gateway* fais confiance pour authentifier des partenaires de transfert.
