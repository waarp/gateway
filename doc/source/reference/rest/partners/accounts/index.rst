Gestion des comptes locaux
##########################

Le point d'accès pour gérer les comptes locaux est ``/api/partners/{partner_name}/accounts``.
Un compte distant est toujours rattaché à un partenaire, et sert donc à s'authentifier
sur ce dernier.
